## 🧠 About This Project

This project is a **frontend UI clone of Facebook**, built to practice:

- Responsive layout design
- Flexbox & positioning
- Sidebar + Navbar structure
- Real-world UI replication

---

## ⚒️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid)  
- JavaScript (basic UI interactions)  
- Font Awesome Icons  

---

## 📸 Features

- 🔵 Facebook-like navbar
- 👤 Left & right sidebars
- 📰 News feed layout
- 📱 Responsive design (mobile support)
- 🎯 Pixel-like UI structure
